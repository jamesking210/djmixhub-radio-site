About DJMixHub

Built for music first, with room to grow.

DJMixHub.com is a simple, stylish radio station website for showcasing live audio, future DJ takeovers, and an organized station schedule. It is designed to stay easy to maintain in GitHub and easy to deploy with Docker.

Always-on player

The player lives at the bottom of the page, so it keeps playing while listeners move between sections on the same page.

AzuraCast-powered data

Listener count, current song details, album art, and schedule data are pulled from AzuraCast so the site reflects the live station.

Ready for future programming

Right now the layout works great with default automation, but it is already structured for scheduled DJs and dedicated shows later.
