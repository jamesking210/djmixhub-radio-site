About DJMixHub

Built for fun, kept simple, made to sound alive.

DJMixHub is an always free radio station. Jimbo built it as a fun way to feature DJs from all around the world and give different mixing styles a home in one place.

The whole point is to keep it real and keep it easy: good mixes, a clean site, no bloated nonsense, and an open door for new people who want to contribute. It runs on Docker and AzuraCast, and the site itself is easy to keep updated on GitHub.

How it’s made

Simple stack. Clean setup. Easy to keep alive.

DJMixHub is built to stay lightweight and practical. The station runs on AzuraCast, the site lives as a simple static front end, and everything is set up so updates are easy to push through GitHub.

The stack

AzuraCast Handles the radio station, streaming, metadata, listener stats, and now-playing feed.
Static site Simple HTML, CSS, and JavaScript so the page stays fast and easy to edit.
Docker Keeps deployment easy on the same machine as the station.
