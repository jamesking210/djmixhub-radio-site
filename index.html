About the station

The story behind DJMixHub.com

Jimbo created DJMixHub.com for fun as a free way to feature DJs from all around the world and show off their different music tastes, mixing styles, and energy.

Built to feel like a real station

The idea was simple: make a front-facing website that looks like a professional radio station page instead of a plain backend control panel. Listeners should be able to press play, explore the station, discover DJs, and keep the music going.

DJMixHub runs on Docker and AzuraCast, which makes it easy to self-host, easy to deploy on nuc1, and easy to keep improving over time.

Open project, easy to improve

The website is maintained as a GitHub project so updates are simple and portable. You can view the repo here: github.com/jamesking210/djmixhub-radio-site.

The live station is powered by AzuraCast at radio.djmixhub.com.
