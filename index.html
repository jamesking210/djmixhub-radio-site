Open Source Energy. Community Sourced Mixes.

Community-sourced mixes, on air.

DJMixHub is a free internet radio station built around long-form DJ mixes. It is open source, self-hosted, and run as a non-commercial project.

The goal is simple: keep good mixes playing, make the station easy to use on phone and desktop, and give more DJs a place to be heard.

DJMixHub runs on a small, reliable stack that stays easy to deploy and self-host. The station uses AzuraCast for radio automation, Docker for deployment, and a lightweight static front end.

AzuraCast Station automation, metadata, listener counts, and public API.
Docker Simple local and self-hosted deployment.
Static site Fast to edit, easy to host, and easy to keep clean.
