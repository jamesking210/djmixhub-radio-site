About the station

Built for fun, built to feature DJs.

Jimbo created this site and station for fun as a free way to feature DJs from all around the world and show off their different music and mixing styles.

How it runs

DJMixHub is hosted with Docker and powered by AzuraCast. The website and station are designed to be simple, fast, and easy to keep online while still feeling like a real branded home for the stream.

The site project is also available on GitHub here: jamesking210/djmixhub-radio-site

Why it exists

This station is about variety. Different DJs bring different tastes, different transitions, and different energy. DJMixHub gives those styles one place to live.

It is meant to be fun, open, and always growing.
